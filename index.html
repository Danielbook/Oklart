Oklart

För att testa applikationen måste du ha npm och bower installerat.

Därefter ladda ner .zip-filen, gå in i terminalen och ställ dig i mappen SMHIApp. Kör därefter kommandona:

npm install

npm start

En lokal server startar på datorn, öppna webläsare och skriv url:en http://localhost:3000
